尽管 GitHub Pages 无需域名即可使用，但是如果要迁移至一个新的服务器（或者空间）呢？原有的 `github.io` 就无法保留。

因此，我购入了 `kevinh.wang` 这个域名。

## 含义

我的英文名是 K. HWANG，去年下半年开始使用。在此之前用的是全称——Kevin Hwang。

购入这个域名，也就顺理成章。

我姓黄，而英文 Last Name 是 Hwang，这是韩国「黄」姓的英译。
